## 卷一: 古代哲学家



### 第一篇：前苏格拉底哲学家
> 啊哈! 以苏格拉底作为`anchor point`

1. 哲学是从`泰勒斯`开始的，他预言日蚀的时间是公元前585年。哲学和科学原是不分的，因此它们一起诞生于公元前第6世纪的初期。
2. 埃及人的神学，主要的关怀是死亡，它们相信死者的灵魂要进入阴间，在那里，`奥西里斯`要根据他们在地上的生活方式来审判他们。（公元前4000年）
3. 巴比伦的发展史比埃及更带有黩武好战的特质。“苏玛连”人发明了楔形文字。曾经有一个时期，有许多独立的城邦彼此互相作战；但是最后巴比伦称霸，并且建立了一个帝国。其他城邦的神就变成了附属的神，而巴比伦的神`马尔督克`便获得了有如后来宙斯在希腊众神之中所占的那种地位。在埃及也出现过同样的情形，只是要早得多。
4. 本质上是一种生殖崇拜
